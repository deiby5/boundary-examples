# CV Screener

Scaffold for a Boundary example that will screen resumes against job requirements and validate structured model output.

This example is not implemented yet. The intended layout is ready for Node.js and Python implementations, plus fixtures for resumes, job descriptions, and expected outputs.

## Project Structure

```text
cv-screener/
├── README.md
├── .env.example
├── fixtures/
│   ├── resumes/
│   ├── jobs/
│   └── expected/
├── node/
└── python/
```
