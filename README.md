# Boundary Examples

Example projects that demonstrate how to use Boundary contracts to validate and repair AI outputs.

## Examples

| Example | Description | Implementations |
|---|---|---|
| [Receipt Scanner](examples/receipt-scanner/) | Extracts structured receipt data from receipt images and validates model output with Boundary. | Node.js, Python |
| [CV Scanner](examples/cv-scanner/) | Extracts structured resume data from PDFs and validates model output with Boundary. | Node.js, Python |

Each example is self-contained and includes its own setup instructions, fixtures, and language-specific implementation folders.
