# Boundary Examples

Example projects that demonstrate how to use Boundary contracts to validate and repair AI outputs.

## Examples

| Example | Description | Implementations |
|---|---|---|
| [Receipt Scanner](examples/receipt-scanner/) | Extracts structured expense data from receipt images and validates model output with Boundary. | Node.js, Python |
| [CV Screener](examples/cv-screener/) | Scaffold for a resume screening example. | Node.js, Python |

Each example is self-contained and includes its own setup instructions, fixtures, and language-specific implementation folders.
